## Resolution Guidelines

When multiple skills compete, prefer the more **specific** one:

- **Animation type known?** → `gsap` (JS tweens), `gsap-scrolltrigger` (scroll-driven), `css-design` (pure CSS), `animation-specialist` (mixed/theory) over each other.
- **SVG with SMIL specifically?** → `svg-graphics` over `smil-animation` (svg-graphics is the superset; smil is legacy/standalone).
- **3D scene vs. VR scene?** → `threejs-3d-webgpu` for raw Three.js; `a-frame` when HTML primitives or WebXR headset context is mentioned.
- **Math concept?** → `geometry` (spatial/game), `linear-algebra` (pure/academic), `trigonometry` (angles/periodic), `game-math` (applied 3D pipeline).
- **Generative art?** → `art-theory` when an art movement is named; `p5js-creative-coding` when p5 or canvas is implied; `animation-specialist` when motion is the focus.
- **Physics?** → `physics` for simulation/integration math; `game-math` for collision/spatial math; `geometry` for shape/volume queries.
